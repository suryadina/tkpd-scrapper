Input Argument:
01. Nama file hasil scrape tokopedia
02. Nama file output hasil pengolahan script ini
03. Humanprint: if set True, program will read comparation logfile (generated by tokopedia-product-data comparator.py)  and use it as an input filter for read-friendly text file output of this program.

Catatan:
01. Script ini membutuhkan file csv dengan format yang sesuai dengan hasil dari script "scrap-produk-tokopedia"
02. Sistem markup harga dari supplier yang digunakan script ini di hard-code. jadi kalo ingin diubah harus diubah langsung di sini.
03. Dalam script ini diasumsikan bahwa nama file gambar untuk setiap produk adalah "standar" sesuai dengan output dari script "scrap-produk-tokopedia"
04. Dalam script ini juga diasumsikan bahwa untuk setiap produk terdapat 5 gambar. Meskipun dalam praktiknya biasanya jarang yang sampai ada 5 gambar.
05. Penggunaan nama gambar yang tidak "standar" berarti harus mengubah manual output dari script ini.
06. Nama file gambar yang diisikan 5 untuk semua produk harusnya tidak jadi masalah. karena dalam  proses upload ke tokopedia atau web tidak akan bikin error. jadi gak perlu dihapus gambar yang kelebihan juga gapapa
